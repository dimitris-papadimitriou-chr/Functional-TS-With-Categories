## using VS Code

 1. open a terminal and type : ***npm install @types/node*** in order to install the node typings 
 2. press ***Ctr+Shift+B*** to display the **tsc:watch** task from the menu on top in order to build and watch for changes 
 3. press ***F5 to debug***
